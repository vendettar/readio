# Cloud 前端

这个目录用于存放 `apps/cloud-ui` 的长期前端实现说明。

`cloud/handoff` 负责 ownership、后端边界与 rollout contract；
`cloud/frontend` 负责浏览器本地 UI 行为、持久化预期与页面级前端机制。

## 当前范围

- `frontend/i18n.zh.mdx`
- `frontend/audio-engine.zh.mdx`
- `frontend/transcript-reading.zh.mdx`
- `frontend/files-management.zh.mdx`
- `frontend/settings.zh.mdx`
