---
title: Playback History
---

## 1. Scope

- Route: `/history`
- Main authority: `apps/cloud-ui/src/store/historyStore.ts`
- Persistence table: browser-local playback sessions

## 2. Contract

- Remote playback writes must keep `countryAtSave`
- For navigable remote/library-origin history records, `countryAtSave` is part of route identity. History deep links must reuse the saved country instead of substituting the current global content region.
- History remains browser-local
- Deleting history removes sessions, not the underlying downloaded/local media artifacts
