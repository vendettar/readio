# 排印与可读性标准

Readio 是一个阅读工具。阅读体验的质量是我们的核心竞争优势。

## 1. 黄金比例

- **行高**: 逐字稿区域的默认值必须是 **1.6**。
  - 为什么? 防止阅读时“跳行”。
- **段落间距**: 使用 **1.2em** 到 **1.5em**。
  - 为什么? 在对话轮次之间提供清晰的视觉中断。
- **行长**: 理想的阅读宽度是 **60-80 字符** (大约 600px - 800px)。
  - 为什么? 长行会导致眼睛疲劳。

## 2. 动态缩放策略

**规则**: 缩放必须是“流动的” (重新排列文本)，而不仅仅是视觉缩放。
- **机制**: 调整容器的 `font-size`。
- **约束**: **不要** 对主要阅读区域使用 `transform: scale()`，因为它会产生模糊的文本和破坏的滚动条。

## 3. 阅读主题 (对比度)

每个主题必须符合 **WCAG AA** (4.5:1) 标准。

| 主题 | 背景 | 前景 | 意图 |
|---|---|---|---|
| **Default** | `var(--background)` | `var(--foreground)` | 系统标准 |
| **Sepia** | `#f4ecd8` | `#5b4636` | 温暖，减少蓝光 |
| **Paper** | `#f5f5f5` | `#333333` | 类似印刷品的清晰度 |
| **Slate** | `#0f172a` | `#f1f5f9` | 高对比度深色 |

## 4. 字体

- **主要**: 系统 UI 堆栈 (`-apple-system`, `SF Pro`, `Inter`)。
- **衬线 (可选)**: 为长篇阅读提供衬线选项 (`Georgia`, `Charter`)。
- **阅读障碍友好**: 支持 `OpenDyslexic` 作为面向未来的选项。
