# 排印与可读性标准

Readio 是一个阅读工具。阅读体验的质量是我们的核心竞争优势。

## 1. 黄金比例（已实现）

- **行高**: 通过 `--reading-line-height` 固定为 **1.6**。
- **段落间距**: 通过 `--reading-para-spacing` 固定为 **1.2em**。
- **行长**: 容器最大宽度限制为 `3xl` (约 768px)，以保持最佳专注度。

## 2. 动态缩放策略（已实现）

**规则**: 缩放是“流动的” (重新排列文本)，由具备**区域特定基准大小**的 CSS 变量控制：
- **机制**: `:root` 上的 `--zoom-scale` 变量联动更新。
- **常规文本 (Prose)**: 使用 `14px` 基准 (`calc(0.875rem * var(--zoom-scale))`)。
- **逐字稿 (Reading)**: 使用 `20px` 基准 (`calc(1.25rem * var(--zoom-scale))`)。
- **约束**: 由于字体文件本身在新尺寸下重新渲染，每个单词都保持绝对清晰。

## 3. 阅读主题 (对比度)

每个主题必须符合 **WCAG AA** (4.5:1) 标准。

| 主题 | 背景 | 前景 | 意图 |
|---|---|---|---|
| **Default** | `var(--background)` | `var(--foreground)` | 系统标准 |
| **Sepia** | `#f4ecd8` | `#5b4636` | 温暖，减少蓝光 |
| **Paper** | `#f5f5f5` | `#333333` | 类似印刷品的清晰度 |
| **Slate** | `#0f172a` | `#f1f5f9` | 高对比度深色 |

## 4. 字体

- **主要**: 系统 UI 堆栈 (`-apple-system`, `SF Pro`, `Inter`)。
- **衬线 (可选)**: 为长篇阅读提供衬线选项 (`Georgia`, `Charter`)。
- **阅读障碍友好**: 支持 `OpenDyslexic` 作为面向未来的选项。
