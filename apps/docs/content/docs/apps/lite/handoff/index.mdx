## 2. What the Project Does (Product/Functional Overview)

Readio is an in-browser podcast player supporting:
- **Files**: Drag and drop MP3 + SRT subtitle files for playback and reading, with folder management support.
- **Explore**: Global search, podcast search, podcast subscription, and episode favoriting.
- **Subtitle Following**: Subtitles automatically scroll and highlight during playback.
- **Selection & Lookup**: Select subtitle text to query dictionaries/web search.
- **Dictionary Pause Setting**: Settings now expose `pauseOnDictionaryLookup` across all supported locales, with fail-closed default `true`.
- **Global Search**: Top sidebar search, supporting secondary modes (History, Discover, Local) and playback status badges.

Main Entry Points (Routes):
- `/` - Main Player Page (Subtitle Reading)
- `/search` - Global Search Results Page
- `/explore` - Explore Page (Find Podcasts)
- `/subscriptions` - Subscriptions List
- `/favorites` - Favorite Episodes
- `/history` - Playback History
- `/files` - Files Page (Audio/Subtitle management, Folder support, Drag & Drop)
- `/settings` - Settings Page
