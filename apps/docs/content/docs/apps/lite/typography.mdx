# Typography & Readability Standards

Readio is a reading tool. The quality of the reading experience is our primary competitive advantage.

## 1. The Golden Ratios (Implemented)

- **Line Height**: Fixed at **1.6** for the transcript area via `--reading-line-height`.
- **Paragraph Spacing**: Fixed at **1.2em** via `--reading-para-spacing`.
- **Line Length**: Container max-width is constrained to `3xl` (approx 768px) for optimal focus.

## 2. Dynamic Zoom Strategy (Implemented)

**Rule**: Scaling is "Fluid" (re-flowing text), controlled by CSS variables with **area-specific base sizes**:
- **Mechanism**: The `--zoom-scale` variable is updated on the `:root`.
- **Prose (Episode Info)**: Uses a `14px` base (`calc(0.875rem * var(--zoom-scale))`).
- **Transcript (Reading)**: Uses a `20px` base (`calc(1.25rem * var(--zoom-scale))`).
- **Constraint**: Every word remains razor-sharp as the font file itself is re-rendered at the new size.

## 3. Reading Themes (Contrast)

Every theme must meet **WCAG AA** (4.5:1) standards.

| Theme | Background | Foreground | Intent |
|---|---|---|---|
| **Default** | `var(--background)` | `var(--foreground)` | System standard |
| **Sepia** | `#f4ecd8` | `#5b4636` | Warm, reduces blue light |
| **Paper** | `#f5f5f5` | `#333333` | Print-like clarity |
| **Slate** | `#0f172a` | `#f1f5f9` | High contrast dark |

## 4. Fonts

- **Primary**: System UI stack (`-apple-system`, `SF Pro`, `Inter`).
- **Serif (Optional)**: Provide a Serif option (`Georgia`, `Charter`) for long-form reading.
- **Dyslexic-friendly**: Support `OpenDyslexic` as a future-proof option.
