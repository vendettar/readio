## 12. 未来阶段：“混合后端” (v2.0)
**目标**：为高级用户提供可选的云同步。

### 架构变更
```mermaid
graph TD
    A[Vite Client] -->|REST| B[Spring Boot API]
    A -->|Direct| C[IndexedDB (Offline Cache)]
    B --> D[PostgreSQL]
```

### 关键模块
1. **认证**: 基于 JWT 的无状态认证。
2. **同步引擎**: “最后写入者胜”同步策略。
3. **CORS 代理**: 自托管代理。
