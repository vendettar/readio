# Product Feature Backlog & Ideas

This document serves as a holding area for future feature ideas, user requests, and non-critical enhancements.
Items here are **not yet scheduled** for development.

---

## üéß Audio & Playback

### 1. Sleep Timer (P0)
- **Goal**: Stop playback after X minutes or end of episode.
- **Complexity**: Low.
- **Implementation**: `useSleepTimer` hook + Store state.

### 2. Smart Speed / Silence Skipping (P1)
- **Context**: Podcast players often skip silence to save time.
- **Goal**: Implement client-side silence detection or use metadata to skip gaps.
- **Complexity**: High (Audio processing).

### 3. Podcast Chapters (P0)
- **Description**: Support `<podcast:chapters>` JSON/PSC.
- **Goal**: Show chapter markers on progress bar and list.
- **Complexity**: Medium (Parsing + UI).

### 4. Playlist Persistence (P1)
- **Context**: Currently, the "Up Next" queue is in-memory (Zustand).
- **Goal**: Persist the current queue to Dexie.
- **Complexity**: Medium.

## üöÄ Performance & PWA

### 1. Offline Mode (P0)
- **Goal**: Cache audio files and allow playback without network.
- **Tech**: Service Worker + Cache Storage API.

### 2. Web Worker Offloading (P1)
- **Goal**: Move RSS parsing and File ID3 reading to a Worker.
- **Benefit**: Unblock main thread during heavy imports.

### 3. Media Cache Eviction Policy (P1)
- **Goal**: Auto-trim audio blobs with LRU strategy.
- **Benefit**: Prevent storage exhaustion.
