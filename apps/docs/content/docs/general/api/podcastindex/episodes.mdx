# Episode Endpoints (/api/1.0/episodes)

Accessing episode-level data and discovery. **Requires Authentication.**

## `/byfeedid`
Retrieve the complete episode list for a show. This is the primary way to pull show content.
- **URL**: `/api/1.0/episodes/byfeedid?id=75075&max=20`

## `/byfeedurl`
Retrieve episode list via the show's RSS URL.
- **URL**: `/api/1.0/episodes/byfeedurl?url={url}`

## `/byitunesid`
Retrieve episode list by iTunes podcast ID.
- **URL**: `/api/1.0/episodes/byitunesid?id=1441923632`

## `/byguid`
Lookup a specific episode by its persistent RSS `<guid>`.
- **URL**: `/api/1.0/episodes/byguid?guid=incomparable:batmanuniversity:41`
- **Example Response**:
```json
{
  "status": "true",
  "episode": {
    "id": 16795088,
    "title": "Gotham",
    "description": "...",
    "enclosureUrl": "https://www.theincomparable.com/podcast/batmanuniversity302.mp3",
    "feedId": 75075
  }
}
```

## `/byid`
Lookup a specific episode by its internal Index ID.
- **URL**: `/api/1.0/episodes/byid?id=16795090`

## `/bysoundbiteid`
Lookup episode by soundbite ID.
- **URL**: `/api/1.0/episodes/bysoundbiteid?id={id}`

## `/random`
Returns random episodes. Use the `max` and `lang` parameters for better results.
- **URL**: `/api/1.0/episodes/random?max=5`
- **Example Response**:
```json
{
  "status": "true",
  "episodes": [
    {
      "id": 16795088,
      "title": "Gotham",
      "feedTitle": "Batman University"
    }
  ]
}
```

## `/live`
Retrieve active `<podcast:liveItem>` streams for shows currently broadcasting.
- **URL**: `/api/1.0/episodes/live`
- **Example Response**:
```json
{ "status": "true", "items": [{ "id": 10001, "title": "Batman Live", "status": "live", "feedId": 75075 }] }
```
