# Episode Endpoints (/api/1.0/episodes)

Accessing episode-level data and discovery. **Requires Authentication.**

## `/byfeedid`
Retrieve the complete episode list for a show. This is the primary way to pull show content.
- **URL**: `/api/1.0/episodes/byfeedid?id=75075&max=20`

## `/bypodcastguid`
Retrieve episode list by the show's [Podcast GUID](https://github.com/Podcastindex-org/podcast-namespace/blob/main/docs/1.0.md#guid).
- **URL**: `/api/1.0/episodes/bypodcastguid?guid=856cd618-7f34-57ea-9b84-3600f1f65e7f&pretty`
- **Example Response**:
```json
{
  "status": "true",
  "items": [
    {
      "id": 16795088,
      "title": "Gotham",
      "podcastGuid": "856cd618-7f34-57ea-9b84-3600f1f65e7f"
    }
  ],
  "count": 1
}
```

## `/byguid`
Lookup a specific episode by its persistent RSS `<guid>`.
- **URL**: `/api/1.0/episodes/byguid?guid=incomparable:batmanuniversity:41&feedid=75075&pretty`
- **Example Response**:
```json
{
  "status": "true",
  "id": "75075",
  "url": "https://feeds.theincomparable.com/batmanuniversity",
  "guid": "incomparable:batmanuniversity:41",
  "episode": {
    "id": 16795088,
    "title": "Gotham",
    "enclosureUrl": "https://www.theincomparable.com/podcast/batmanuniversity302.mp3",
    "feedId": 75075
  }
}
```

## `/byid`
Lookup a specific episode by its internal Index ID.
- **URL**: `/api/1.0/episodes/byid?id=16795090&pretty`
- **Example Response**:
```json
{
  "status": "true",
  "id": 16795090,
  "episode": {
    "id": 16795090,
    "title": "A Different Kind of Hero",
    "feedId": 75075
  }
}
```


## `/random`
Returns random episodes. Use the `max` and `lang` parameters for better results.
- **URL**: `/api/1.0/episodes/random?max=2&pretty`
- **Example Response**:
```json
{
  "status": "true",
  "episodes": [
    {
      "id": 16795088,
      "title": "Gotham",
      "feedTitle": "Batman University"
    }
  ],
  "count": 1,
  "max": "2"
}
```

## `/live`
Retrieve active `<podcast:liveItem>` streams for shows currently broadcasting.
- **URL**: `/api/1.0/episodes/live?pretty`
- **Example Response**:
```json
{
  "status": "true",
  "items": [
    {
      "id": 16795088,
      "title": "Live Stream Name",
      "status": "live"
    }
  ],
  "count": 1
}
```
