# Static Data

The Podcast Index provides static data files for large-scale analysis or bulk processing.

## `dead_feeds.csv`
A list of all feeds in the index that have been identified as inactive or dead.
- **Location**: Indexed periodically and available for bulk download.

## Local Data
For large integrations, it is often recommended to cache results or download the full index database snapshot if allowed by current API policies.
