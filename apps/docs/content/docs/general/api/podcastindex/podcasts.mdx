# Podcast (Feed) Endpoints (/api/1.0/podcasts)

Retrieving specific podcast feed data and show discovery. **Requires Authentication.**

## `/byfeedurl`
Lookup show metadata via its RSS URL.
- **URL**: `/api/1.0/podcasts/byfeedurl?url={url}`
- **Example Response**:
```json
{ "status": "true", "feed": { "id": 75075, "podcastGuid": "9b024349-...", "title": "Batman University", "url": "https://feeds.theincomparable.com/batmanuniversity" } }
```

## `/byitunesid`
Lookup using the Apple store ID.
- **URL**: `/api/1.0/podcasts/byitunesid?id=1441923632`
- **Example Response**:
```json
{ "status": "true", "feed": { "id": 75075, "title": "Batman University", "itunesId": 1441923632 } }
```

## `/byguid`
Lookup via its Podcasting 2.0 global ID.
- **URL**: `/api/1.0/podcasts/byguid?guid=917393e3-1b1e-5cef-ace4-edaa54e1f810`
- **Example Response**:
```json
{ "status": "true", "feed": { "id": 75075, "podcastGuid": "9b024349-ccf0-5f69-a609-6b82873eab3c", "title": "Batman University" } }
```

## `/bytag`
Filter podcasts by tags (e.g., `podcast-value`).
- **URL**: `/api/1.0/podcasts/bytag?q=podcast-value`
- **Example Response**:
```json
{ "status": "true", "feeds": [{ "id": 75075, "title": "Batman University" }], "count": 1 }
```

## `/trending`
Returns currently trending podcasts. Supports `lang`, `cat`, and `notcat` filters.
- **URL**: `/api/1.0/podcasts/trending`
- **Example Response**:
```json
{ "status": "true", "feeds": [{ "id": 75075, "title": "Trending Show", "trendScore": 12 }] }
```

## `/dead`
Shows marked as dead or inactive.
- **URL**: `/api/1.0/podcasts/dead`

## `/newfeeds`
Newly added shows to the index.
- **URL**: `/api/1.0/podcasts/newfeeds`
