# Glossary

## Discovery
The podcast discovery surface (Explore) powered by provider APIs (charts, picks, search).

## Global Search
App-wide search system with an overlay preview and a full results page.

## Hybrid ID
Episode URL format `{readable-title}-{first8}` where `first8` is the first 8 chars of the GUID.

## Singleton Playback
Only one tab can play audio at a time; other tabs must pause.
