# Patch Policy

# Patch Notes Usage

These patch files are **review notes / improvement suggestions**, not automatic rules.

## Canonical SSOT
When patch notes conflict with existing documentation, **the SSOT is**:
- `apps/docs/content/docs/apps/lite/handoff/*`
- `apps/docs/content/docs/general/*`
- Design system docs

## How to Use
- Only apply patches that **do not conflict** with SSOT.
- Treat patches as **execution requirements** only when explicitly approved.
- Do **not** mass‑edit historical instructions unless they are actively referenced.
