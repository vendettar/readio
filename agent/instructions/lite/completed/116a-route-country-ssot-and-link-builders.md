# Task: 116a (Patch) - Route Country SSOT + Central Link Builders [COMPLETED]

## Goal
Make route country (`/$country/...`) the single authority for podcast content routes, and centralize all podcast/show/episode link construction.

## Scope Scan (8 Scopes)
- Config: no new env keys.
- Persistence: no schema change.
- Routing: add country-prefixed route tree.
- Logging: debug-level only for invalid country canonicalization.
- Network: no endpoint changes.
- Storage: no migration in this step.
- UI state: remove country authority from state for content-route correctness.
- Tests: route validation + link generation + topology continuity.

## Hidden Risk Sweep
- Async: ensure `beforeLoad` redirects only; no render-phase redirect loops.
- Hot-path: avoid runtime country inference in route components.
- State transition integrity: invalid country/slug always has deterministic redirect.
- Dynamic context consistency: changing global Explore country must not mutate current `/$country/...` behavior.

## Required Patterns
- Use one country whitelist source: `SUPPORTED_CONTENT_REGIONS` from `apps/lite/src/constants/app.ts`.
- Parent route `/$country` is the only country validator/canonicalizer.
- Podcast route links must be generated by builder helpers (no inline string path assembly).

## Forbidden Dependencies
- No new routing framework.
- No compatibility adapters for old `/podcast/*` paths in this step.

## Implementation Steps
1. Add route parent guard:
   - `apps/lite/src/routes/$country.tsx`
   - Validate country against `SUPPORTED_CONTENT_REGIONS`.
   - Canonicalize to lowercase in `beforeLoad`.
   - Reject unsupported country in `beforeLoad` (no render-phase fixup).

2. Add country-scoped route tree:
   - `apps/lite/src/routes/$country/podcast/$id.tsx`
   - `apps/lite/src/routes/$country/podcast/$id/index.tsx`
   - `apps/lite/src/routes/$country/podcast/$id/episodes.tsx`
   - `apps/lite/src/routes/$country/podcast/$id/episode/$episodeId.tsx`
   - Keep old `/podcast/*` files temporarily until 116c completion.

3. Add centralized route builder module:
   - `apps/lite/src/lib/routes/podcastRoutes.ts`
   - Required APIs:
     - `normalizeCountryParam(country): SupportedCountry | null`
     - `buildPodcastShowRoute({ country, podcastId, search? })`
     - `buildPodcastEpisodeRoute({ country, podcastId, episodeSlug, search? })`
   - Return TanStack-compatible objects (`to`, `params`, optional `search`) only.
   - `normalizeCountryParam` must not fallback to global/default country.

4. Migrate all active link producers to builder APIs:
   - `apps/lite/src/routeComponents/HistoryPage.tsx`
   - `apps/lite/src/routeComponents/FavoritesPage.tsx`
   - `apps/lite/src/routeComponents/SubscriptionsPage.tsx`
   - `apps/lite/src/components/EpisodeRow/EpisodeRow.tsx`
   - `apps/lite/src/components/Explore/PodcastEpisodesGrid.tsx`
   - `apps/lite/src/routeComponents/SearchPage.tsx`
   - `apps/lite/src/components/GlobalSearch/*`
   - `apps/lite/src/lib/localSearchActions.ts`
